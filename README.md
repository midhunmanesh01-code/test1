# 🚀 Publish a Folder/File to GitHub (Using VS Code)

## 📌 Steps to Publish


# Initialize git
```bash
git init
```

# Add all files
```bash
git add .
```

# Commit files
```bash
git commit -m "Initial upload"
```

# Create a repository on GitHub (do this manually on GitHub website)

# Connect local repo to GitHub
```bash
git remote add origin https://github.com/username/repositoryname.git
```

# Set main branch
```bash
git branch -M main
```

# Push to GitHub
```bash
git push -u origin main
```
